Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <B> (que é uma tag não semântica).

Nesta frase, tenho um termo em destaque usandi a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usandoa tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Se quiser mudar a cor do marca texto para uma outra cor como essa, basta usar o parametro de CSS "style + background-color" dentro da tag

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

A tag <big> está obsoleta por não fazer mais sentido aumentar a letra nesse tamanho. Poderiamos simplesmente aumentar o tamanho da fonte ou da caixa no caso

A tag <small> ainda é usada por fazer sentido como usamos para ler letras míudas como essa aqui, tipo em um contrato

Texto deletado

Podemos marcar um teto como excluído para indicar que ele deve ser lido, mas nÃo considerado. Simplificando é o famoso texto com risquinho.

Para colocarmos esse risquinho no texto, que damos o nome de "deletado", usamos a tag <del> no texto selecionado. EX: esse texto pode ter sido inválido.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. O famoso texto sublinhado

Para inserir um texto e dar ênfase a ele usamos a tag <ins<. EX: Essa parte AQUI DO TEXTO foi usado a tag <ins>.

A forma não semântica usada é com a tag <u>. Ficando assim.
